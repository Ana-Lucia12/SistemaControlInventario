## Descripción

El sistema permite registrar, editar, eliminar, buscar, filtrar, ordenar y exportar productos de un inventario mediante una aplicación Java Swing.

## Funcionalidades principales

- Registro de productos.
- Validaciones en capa de negocio.
- Lista de productos en JTable.
- Búsqueda por nombre o código.
- Filtro por categoría.
- Ordenamiento por nombre, precio y cantidad.
- Estadísticas del inventario.
- Historial de acciones.
- Exportación del inventario a archivo CSV.
- Uso de JFrame, JMenuBar, JToolBar y JTabbedPane.

## Uso de Collections

- List para almacenar productos.
- Set para evitar códigos duplicados.
- Map para contar productos por categoría.
- Stack para guardar historial de acciones.
