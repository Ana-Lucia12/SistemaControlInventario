Descripción:
Este proyecto consiste en una aplicación Java Swing para administrar productos de inventario.

Funcionalidades principales:
- Registrar productos.
- Editar productos.
- Eliminar productos con confirmación.
- Mostrar productos en una tabla JTable.
- Buscar productos por nombre o código.
- Filtrar productos por categoría.
- Ordenar productos por nombre, precio y cantidad.
- Mostrar estadísticas del inventario.
- Mostrar historial de acciones.
- Exportar el inventario a un archivo.

Capas utilizadas:
- modelo
- repositorio
- negocio
- presentacion
- excepciones
- util

Collections utilizadas:
- List para almacenar productos.
- Set para evitar códigos duplicados.
- Map para contar productos por categoría.
- Stack para guardar el historial de acciones.
